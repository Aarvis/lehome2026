# Assets

Place public-facing media here.

Suggested folders:

- `figures/`: flowcharts, screenshots, rollout strips, diagrams.
- `videos/`: short MP4/WebM clips for sim and real robot setups.
- `papers/`: PDF draft, arXiv source, BibTeX, supplementary notes.
- `tables/`: exported CSV files and plot source data.

Keep filenames lowercase and descriptive.
